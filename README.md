# Geiger Counter

## Hardware Setup
- Geiger Counter
- [ESP32 Dev Board](https://www.az-delivery.de/products/esp32-developmentboard)
- LCD Display
- RTC Module
- SD Card Module + SD Card, notice that the SD Card must be formatted as FAT with max. 2GB partition.
